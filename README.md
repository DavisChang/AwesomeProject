## Testing your React Native Installation (Mac)

Start ios: 

 `react-native run-ios`

Start android: 

 - List all your emulators  
 	`emulator -list-avds`
 	
 - Start emulator	
 	`emulator -avd @name-of-your-emulator`
 	
 - Runing App on the Android emulator   
	`react-native run-android`
